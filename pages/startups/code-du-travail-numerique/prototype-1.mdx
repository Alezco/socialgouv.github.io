# Code du travail numérique

## Premières recherches

### Prototype #1

Nous rassemblons toutes les données disponibles et créons une première interface de navigation basé sur l'arborescence des tags du logiciel ePoseidon (à destination des inspecteurs du travail).

![resultats recherche](/static/images/startups/code-du-travail-numerique/cdtn-themes.png)

L'interface regroupe toutes les sources de données pour un tag ePoseidon donné :

![resultats recherche](/static/images/startups/code-du-travail-numerique/cdtn-resultats.png)

Nous faisons plusieurs itérations sur ce prototype et améliorons la page de résultat en prenant en compte l'articulation des normes et en ajoutant la recherche de convention collective et la localisation du [service de renseignement](/startups/code-du-travail-numerique/sdr) le plus proche.

Le résultat est visible sur https://socialgouv.github.io/code-du-travail-explorer/

#### Enseignements :

 - La structure de **l'arborescence des tags est déroutante pour un novice** du droit car ce sont des termes métiers
 - L'utilisateur ne sait pas forcement ce qu'il cherche exactement, donc se trouve bloqué rapidement
 - Côté affichage des résultats, il faudra privilégier les sources en "Français facile"

 ➡️ Le système de navigation est à revoir

 ➡️ L'appui d'un(e) spécialiste UX sera grandement utile pour tester les hypothèses


#### Hypothèses :

  - Ce mode de navigation pourrait fonctionner avec des tags plus "user-friendly".
  - Un champ de recherche + affinage si nécessaire serait plus adapté

### Autres pistes

D'autres pistes ont été explorées :

 - divers essais d'exploitation de la base KALI
