# Code du travail numérique

## Les sources de données

Parallèlement aux [immersions](./kickoff), une des premières missions a été de cartographier les données existantes.

![sources de données](/static/images/startups/code-du-travail-numerique/cdtn-sources.png)

| source                     | accès                                                                                                                            |
|:---------------------------|:---------------------------------------------------------------------------------------------------------------------------------|
| Code du travail            | [site legifrance](https://www.legifrance.gouv.fr/affichCode.do?cidTexte=LEGITEXT000006072050)                                    |
| Code du travail + tags     | logiciel ePoseidon (DGT)                                                                                                         |
| Conventions Collectives    | logiciel BD3C (DGT) ou [base KALI](https://www.data.gouv.fr/fr/datasets/kali-conventions-collectives-nationales/)                                                                                                             |
| Accords d'entreprise       | logiciel NgAccord (DGT)                                                                                                          |
| FAQ                        | Services de renseignement                                                                                                        |
| Liens service-public       | [data.gouv.fr](https://www.data.gouv.fr/fr/datasets/service-public-fr-guide-vos-droits-et-demarches-professionnels-entreprises/) |
| Annuaire géo SDR           | [data.gouv.fr](https://www.data.gouv.fr/fr/datasets/service-public-fr-annuaire-de-l-administration-base-de-donnees-locales/)     |
| Liens ministère-du-travail | scrapping                                                                                                                        |
| liens et outils            | manuel                                                                                                                           |

Nous indexons ces données dans un serveur ElasticSearch (solution plébiscitée par tous nos interlocuteurs) pour commencer à pouvoir faire des recherches et optimiser les résultats.



Dans le domaine du droit libre, nous avons découvert de très beaux projets :

| projet                                                         | description                                                        |
|:---------------------------------------------------------------|:-------------------------------------------------------------------|
| [LegiLibre](https://github.com/legilibre)                      | suite d'outils autour de la [base LEGI](https://www.data.gouv.fr/fr/datasets/legi-codes-lois-et-reglements-consolides/)                              |
| [bureau-ouvert](https://parlement-ouvert.fr)                   | outils autour du droit "chaud" (assemblée nationale, parlement...) |
| [regards-citoyens](https://www.regardscitoyens.org/#&panel1-1) | la fabrique de la loi, [nosdeputes.fr](http://www.nosdeputes.fr)                                 |

Et également des projets connexes chez beta.gouv.fr :

 - [SYSO](https://github.com/betagouv/syso) : le simulateur de salaires
 - [ZAM](https://github.com/betagouv/zam) : la gestion des amendements
